# TripMate-AI

## 🛠️ Tech Stack

### **Frontend**

**Web:** React.js + Next.js + Tailwind CSS

**Maps & Location:** Google Maps API / Mapbox

**Authentication:** Firebase Auth


#### **Backend**

**Server:** Node.js + Express.js

**Database:** PostgreSQL (Supabase)

**Chat & Real-time Data:** Firebase Firestore
